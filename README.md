# mdb (mdb)

Mobile Data Bank

## Install the dependencies
```bash
yarn
```

### Start the app in development mode (hot-code reloading, error reporting, etc.)
```bash
quasar dev
```


### Build the app for production

### 發佈
```bash
quasar build -m pwa
firebase deploy --only hosting
```
###預覽
```
firebase hosting:channel:deploy modatabank
```
